# Getting API tokens for messengers

## VK

1. Create VK group.
2. Go to settings.
3. Go to extra settings.
4. Go to API section.
5. Create API key with `messages` permissions.
6. Create token for authenticated user in messaging service with access_token as given (from vk) token.
